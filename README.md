# 🎯 Number Guessing Game (Java)

A fun and interactive **Java console-based game** where the user tries to guess a randomly generated number within a selected difficulty range.

---

## 📌 Features

- 🎮 Difficulty Levels:
  - Easy (1 - 50)
  - Medium (1 - 100)
  - Hard (1 - 200)
- 🔢 Random number generation using `Random`
- 📥 Input validation (prevents alphabet input)
- 🔁 Option to play multiple rounds
- 📊 Tracks number of attempts
- 🧠 Provides hints:
  - Too High
  - Too Low
  - Correct Guess

---

## 🛠️ Technologies Used

- Java (Core Java)
- OOP Concepts
- `Scanner` for input handling
- `Random` for number generation
